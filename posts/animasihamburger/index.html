Animasi Hamburger sederhana

Dikirim oleh dei pada 2022-06-20
1 minute read
programing
coding , css , Javascript , html

apa itu?

Animasi hamburger yang biasanya digunakan untuk bagian navigasi bar, dalam pembuatannya saya hanya menggunakan html, Javascript, dan css.

Klik dua garis dibawah ini.

Kode nya

Menurut saya ini code sederhana jadi tidak terlalu sulit.

html

  <!-- ###Kode HTML -->
  <div class="burger__body">
    <div class="burger" id="burger" onclick="buka()">
      <div class="line__1" id="line__1"></div>
      <div class="line__2" id="line__2"></div>
    </div>
  </div>
  <!-- ###Kode HTML -->

css

<!-- ###Kode CSS -->
  <style>
    * {
      padding: 0;
      margin: 0;
    }
    .burger__body {
      height: 200px;
      width: 100%;
      display: flex;
      justify-content: center;
      align-items: center;
    }
    .burger {
      width: 200px;
      height: 200px;
      display: flex;
      flex-direction: column;
      justify-content: center;
      align-items: center;
      position: relative;
      cursor: pointer;
    }
    .line__1,
    .line__2 {
      background-color: black;
      height: 10px;
      width: 100px;
      transition-duration: 0.5s;
      position: absolute;
    }
    .line__1 {
      margin-bottom: 2em;
    }
    .line__2 {
      margin-top: 2em;
    }
  </style>
  <!-- ###Kode CSS  -->

Javascript

  <!-- ###Kode Script JS -->
  <script>
    var l1 = document.getElementById("line__1");
    var l2 = document.getElementById("line__2");
    function buka() {
      if (l1.style.transform === "rotateZ(45deg)") {
        l1.style.transform = ("rotateZ(0deg)");
        l2.style.transform = ("rotateZ(0deg)");
        l2.style.marginTop = ("2em");
        l1.style.marginBottom = ("2em");
      } else {
        l1.style.transform = ("rotateZ(45deg)");
        l2.style.transform = ("rotateZ(-45deg)");
        l2.style.marginTop = ("0em");
        l1.style.marginBottom = ("0em");
      }
    }
  </script>
  <!-- ###Kode Script JS -->
